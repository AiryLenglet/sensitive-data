SensitiveString should be created from @Sensitive annotated data

Noncompliant Code Example

class Dto {

  private String value;

  public String getValue() {
    return this.value;
  }
}

final var dto = new Dto();
new SensitiveString(dto.getValue());  // Noncompliant; SensitiveString is produced with non Sensitive data

Compliant Code Example

class Dto {

  private String value;

  @Sensitive
  public String getValue() {
    return this.value;
  }
}

final var dto = new Dto();
new SensitiveString(dto.getValue());  // Compliant
