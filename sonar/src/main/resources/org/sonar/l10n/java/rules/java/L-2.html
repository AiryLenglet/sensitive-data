SensitiveString should be created from @Sensitive annotated data

Noncompliant Code Example

class Dto {

  private String value;

  public void setValue(String value) {
    this.value = value;
  }
}
SensitiveString sensitiveString;
final var dto = new Dto();
dto.setValue(sensitiveString.getValue());  // Noncompliant; SensitiveString value is used with non Sensitive data

Compliant Code Example

class Dto {

  @Sensitive
  private String value;

  public void setValue(String value) {
    this.value = value;
  }
}
SensitiveString sensitiveString;
final var dto = new Dto();
dto.setValue(sensitiveString.getValue());  // Compliant
